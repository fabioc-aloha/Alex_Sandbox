## Writing Process

### Daily Writing

1. Set timer for focused writing block (25-50 min)
2. No editing during drafting
3. Track word count
4. Note questions/gaps for later

### Weekly Review

1. Read week's output
2. Light editing pass
3. Update status tracking
4. Plan next week's focus

### Committee Feedback

1. Submit chapter draft
2. Schedule feedback meeting
3. Document feedback in `notes.md`
4. Revise based on feedback
5. Update status
