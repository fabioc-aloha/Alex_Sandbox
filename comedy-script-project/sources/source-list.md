## Quality Tiers

### Tier 1: Use Immediately
- AP, Reuters, BBC
- Major newspaper local sections
- Verified by multiple outlets

### Tier 2: Verify Before Using
- Reddit submissions
- Aggregator sites
- Social media posts

### Tier 3: Never Use Without Verification
- Daily Mail (known for sensationalism)
- Any site you haven't heard of
- Anything that seems too perfect
