# Joke Workshop

Where jokes go to get better.

---

## Joke Writing Process

1. **First Draft** - Get the idea down
2. **Find the Angle** - What's the funniest part?
3. **Trim the Fat** - Shortest path to punchline
4. **Add a Tag** - Can we get a second laugh?
5. **Table Read** - Does it land out loud?
6. **Punch Up** - Make it tighter/funnier

---

## Joke Templates

### The Comparison

```
[NEWS SETUP]
Which is [more/less] [adjective] than [comparison].
```

Example:
> "A man was arrested for stealing a police car. Which is still
> less embarrassing than my Uber rating."

---

### The Callback

```
[NEWS SETUP]
Making it the [superlative] since [earlier reference].
```

Example:
> "A Florida mayor resigned after a scandal. Making it the most
> dignified exit by a Florida politician this week."

---

### The React

```
[NEWS SETUP]
... [pause] [short reaction].
```

Example:
> "Scientists say talking to plants helps them grow."
> "... I mean, same."

---

### The List of Three

```
[NEWS SETUP]
[Normal], [normal], and [absurd].
```

Example:
> "The thief stole a TV, a laptop, and a decorative gourd from 2019."

---

### The Understatement

```
[ABSURD NEWS]
So, a normal [day/week/Tuesday].
```
