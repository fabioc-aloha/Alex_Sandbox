## Development Process

### Pre-Writing
1. ✅ Create project structure
2. ⬜ Define logline and premise
3. ⬜ Develop characters
4. ⬜ Create outline
5. ⬜ Write beat sheet

### Writing
6. ⬜ Write treatment
7. ⬜ First draft
8. ⬜ Self-review
9. ⬜ Table read
10. ⬜ Incorporate feedback

### Revision
11. ⬜ Second draft
12. ⬜ Polish pass
13. ⬜ Final draft
