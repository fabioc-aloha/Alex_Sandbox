# Characters

This folder contains character profiles for the comedy script.

## Character Hierarchy

### Main Characters
- **Protagonist** - The lead character driving the story
- **Antagonist** - The opposing force (person, situation, or self)
- **Love Interest / Best Friend** - Key supporting relationship

### Supporting Characters
- Secondary characters that support the main storyline

### Comic Relief / Ensemble
- Characters primarily for comedic moments

## Comedy Character Archetypes

Consider these classic comedy archetypes:
- **The Straight Man** - Reacts to absurdity
- **The Fish Out of Water** - Doesn't fit the environment
- **The Lovable Loser** - Flawed but sympathetic
- **The Eccentric** - Quirky, unpredictable behavior
- **The Schemer** - Plans that backfire hilariously
- **The Oblivious One** - Unaware of obvious situations
