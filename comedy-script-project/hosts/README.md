## The Dynamic at a Glance

**Maya:** Deadpan, grounds the chaos, delivers the stinger

**Danny:** Animated, genuine reactions, finds the relatability angle

**Together:** Best friends who make absurd news feel like catching up

---

## Signature Phrases

### Maya

- "I'm sorry, WHAT?"
- "Moving on."
- "Okay, so..."

### Danny

- "Honestly? Relatable."
- "Wait, wait, wait—"
- "That tracks."

### Running Bits

- **Pun War**: Maya claims to hate puns, then drops devastating ones
- **Animal Softie**: Danny gets too invested in animal stories
- **Florida Score**: Tracking how many stories come from Florida
- **The Look**: Maya's wordless "you've gone too far" expression

| If you need... | Give it to... |
| -------------- | ------------- |
| The setup | Either (Maya for dry, Danny for excited) |
| The punchline | Usually Maya |
| The reaction | Danny (verbal), Maya (physical/look) |
| The tangent | Danny |
| The save | Maya ("Anyway—") |
| The callback | Danny |

---

## What Makes Maya & Danny Work

1. **Contrast** - Deadpan vs. Animated
2. **Trust** - They never undermine each other
3. **Balance** - Maya grounds, Danny elevates
4. **Chemistry** - Genuine friendship energy
5. **Timing** - Know when to let the other land
