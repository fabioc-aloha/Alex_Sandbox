## Comedy Techniques to Study

- [ ] **Timing** - Pause, pace, rhythm
- [ ] **Misdirection** - Set up expectations, subvert them
- [ ] **Escalation** - Start small, build bigger
- [ ] **Reversal** - Flip the situation
- [ ] **Callback** - Reference earlier jokes
- [ ] **Fish Out of Water** - Characters in wrong environment
- [ ] **Dramatic Irony** - Audience knows more than characters
- [ ] **Physical Comedy** - Visual gags and slapstick
- [ ] **Wordplay** - Puns, double meanings, misunderstandings
- [ ] **Character Comedy** - Humor from who they are
